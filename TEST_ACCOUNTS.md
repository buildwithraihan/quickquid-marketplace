## How to Use

1. Navigate to the login page
2. Click "Show Test Accounts" button
3. Click on any account to auto-fill the login form
4. Click "Sign In" to log in

## Features by Role

### Buyer
- Browse gigs
- Apply to gigs
- View dashboard
- Chat with sellers (after acceptance)

### Seller
- Create gigs
- Manage services
- View hire requests
- Track orders
- Seller dashboard

### Admin
- Approve/reject gigs
- Manage escrow
- Resolve disputes
- Platform metrics

## Notes

- All accounts are verified and active
- All accounts are associated with Jain University
- New account registration is disabled in PoC mode
- Authentication is handled via localStorage (mock system)
